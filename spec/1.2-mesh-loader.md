# 1.2 Mesh Loader

**Phase**: 1 - Foundation
**Week**: 1
**File**: `src/loaders/mesh_loader.cpp`

## Purpose
Load traditional mesh files (OBJ, FBX) and convert to unified Primitive representation.

## Input
- `.obj`, `.fbx`, or other Assimp-supported formats

## Output
- `Scene` object containing `Primitive` structs (one per vertex)

## Requirements

### Must Parse
- Vertex positions
- Vertex normals (or compute from faces)
- Face topology (for normal computation)
- Material properties (optional)

### Must Convert
- Each vertex → Primitive
- Compute local covariance from k-nearest neighbors (k=10-20)
- Set alpha = 1.0 (fully opaque)
- Handle coordinate system conversions (Y-up ↔ Z-up)

## Implementation Notes
- Use Assimp for file loading
- Build k-d tree or spatial hash for neighbor queries
- Covariance = PCA of local neighborhood

## Validation
- Test multi-material meshes
- Verify normals point outward
- Check coordinate system is consistent

## Dependencies
- `Assimp` (mesh loading)
- `Eigen` (covariance computation)
- Spatial data structure (nanoflann or custom)
