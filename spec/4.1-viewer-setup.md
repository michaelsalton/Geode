# 4.1 Viewer Setup

**Phase**: 3 - Visualization
**Week**: 7
**File**: `src/viewer/`

## Purpose
Interactive 3D viewer for inspecting scenes and classification results.

## Library
**Polyscope** - 3D visualization with ImGui integration

## Features

### Scene Display
- Point cloud rendering (one point per primitive)
- Color from SH DC term or default color
- Adjustable point size

### Camera Controls (Polyscope built-in)
- Orbit: drag with mouse
- Pan: Shift + drag
- Zoom: scroll wheel
- Reset view: 'R' key
- Fit to bounds: auto

### UI Elements
- FPS counter
- Primitive count display
- Scene bounds info

## Performance Targets
- 60 FPS for < 500K primitives
- 30 FPS for 500K-1M primitives
- Smooth camera controls

## Implementation
```cpp
#include <polyscope/polyscope.h>
#include <polyscope/point_cloud.h>

void initViewer(const Scene& scene) {
    polyscope::init();
    polyscope::registerPointCloud("scene", positions);
    polyscope::show();
}
```

## Validation
- Launches without errors
- Scene displays correctly
- Matches other 3D viewers (CloudCompare, MeshLab)

## Dependencies
- Polyscope (will auto-download via CMake)
- OpenGL 3.3+
