# 1.1 PLY Loader

**Phase**: 1 - Foundation
**Week**: 1
**File**: `src/loaders/ply_loader.cpp`

## Purpose
Load PLY files containing 3D Gaussian Splats from NeRF reconstructions (nerfstudio/gsplat format).

## Input
- `.ply` file with Gaussian splat data

## Output
- `Scene` object containing `Primitive` structs

## Requirements

### Must Parse
- `x, y, z` - Position (vec3)
- `scale_0, scale_1, scale_2` - Scale parameters
- `rot_0, rot_1, rot_2, rot_3` - Rotation quaternion
- `opacity` - Alpha value
- `f_dc_0, f_dc_1, f_dc_2` - SH DC coefficients (color)
- `f_rest_*` - SH coefficients (optional)

### Must Convert
- Scale + Rotation → 3x3 covariance matrix
- Opacity → alpha (0-1 range)
- Covariance → normal (smallest eigenvector)

## Implementation Notes
- Use `tinyply` or `happly` (header-only)
- Handle both binary and ASCII PLY
- Validate required properties exist

## Validation
- Test on gsplat/nerfstudio output files
- Verify covariance matrix is symmetric positive-definite
- Check all primitives have valid positions

## Dependencies
- `Eigen` (covariance computation)
- `tinyply` or `happly` (PLY parsing)
