# 3.2 Alpha Metrics

**Phase**: 2 - Classification Metrics
**Week**: 5
**File**: `src/metrics/alpha_metrics.cpp`

## Purpose
Analyze opacity consistency to distinguish opaque surfaces from volumetric/fuzzy regions.

## Metrics

```cpp
struct AlphaMetrics {
    float mean;      // Average opacity
    float variance;  // Opacity variation
    float min;       // Minimum opacity
    float max;       // Maximum opacity
};
```

## Algorithm

1. Extract alpha values from all primitives in cluster
2. Compute: `mean = (1/N) Σ αᵢ`
3. Compute: `variance = (1/N) Σ (αᵢ - mean)²`
4. Record: `min = min(αᵢ)`, `max = max(αᵢ)`

## Interpretation

| Metric | Value | Meaning | Optimal For |
|--------|-------|---------|-------------|
| mean | > 0.85 | Opaque surface | Mesh |
| mean | < 0.5 | Translucent/fuzzy | Gaussian |
| variance | < 0.1 | Consistent opacity | Mesh |
| variance | > 0.2 | Variable opacity | Gaussian |

## Use Cases
- Opaque walls/floors: mean ≈ 1.0, variance ≈ 0
- Smoke/fog: mean < 0.5, high variance
- Foliage: variable alpha, high variance

## Validation
- Test on mesh: α_mean ≈ 1.0, α_var ≈ 0.0
- Test on fuzzy Gaussians: α_var > 0.1
